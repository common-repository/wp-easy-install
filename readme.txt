== Description ==

Use WP Easy Install to setup your newly installed WordPress with just a few clicks. The plugin comes with preloaded options, the only thing you need to do is to press the setting on/off and click "Update".

= You can choose multiple options in the process, some of them are: =

* Delete the "Hello Dolly" plugin
* Activate the Akismet plugin
* Delete all the dummy content
* Delete standard theme
* Choose between WWW or non-WWW
* Select timezone

= Furthermore it is possible to select and install one of the preselected plugins in the plugin. The preselected plugins are: =


* Broken Link Checker
* Fast Secure Contact Form
* Jetpack by WordPress.com
* Limit Login Attempts
* Link Juice Keeper
* WP Super Cache
* Yoast SEO
* Social Warfare
* Accelerated Mobile Pages
* Kraken Image Optimizer
* Nginx Helper
* Table of Contents Plus
* WP-PostRatings

== Installation ==

1. Upload `wp-easy-install.zip` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to the plugin interface through Settings -> Wp Easy Install

4. Select and deselect the options you want, and click on the button

5. Enjoy saving a lot of time when you start up a new Wordpress blog!
